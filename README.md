## ⚙️ How Prices Are Calculated

Gold prices are derived from:

- Spot gold price (USD per ounce)
- USD → GBP exchange rate
- Conversion from ounces to grams (31.1034768g per troy ounce)
- Purity adjustment (based on carat)
